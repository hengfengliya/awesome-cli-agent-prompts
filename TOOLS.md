# TOOLS.md - 本地环境备忘

Skills 定义“怎么用工具”。这个文件记录**你这台环境独有**的信息（不适合放进通用 skill 里）。

## 这里应该写什么

比如：

- 摄像头名称与位置
- SSH 主机别名/地址
- TTS 偏好的声音
- 音箱/房间名称
- 设备昵称
- 任何与环境强绑定的细节

## 示例

```markdown
### 摄像头

- living-room → 客厅主区域，180° 广角
- front-door → 入户门口，支持动作触发

### SSH

- home-server → 192.168.1.100, user: admin

### TTS

- 偏好音色: "Nova"（温暖、略英式）
- 默认播放设备: Kitchen HomePod
```

## 为什么要单独放这里？

Skills 是可共享/可升级的；你的环境是私有且变化的。分开保存可以：
- 更新 skills 时不丢本地配置
- 分享 skills 时不泄露你的基础设施细节
